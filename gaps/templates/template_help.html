Help

Gene App Scraper (GAPS) is a web-application made to update gene panels by discovering genes related to specific diseases and disease characteristics. This can be done by entering various search terms, which lead to obtaining corresponding articles. Out of these articles, multiple results can be found such as gene symbols and diseases. In addition to that, results revolving the article are also gathered, such as: Publication date, DOI, Pubmed id, title and abstract. 

1. The Query_builder

The aim of this page is to create search terms to eventually find corresponding articles. This page provides 3 main elements for creating a search:

  1. A query builder
  2. Specify gene symbols
  3. Optional options

From these search terms multiple articles can be obtained from Pubmed, out of which the contents are searched for possible genes that are related to specific diseases.

1.1. Build a query

The query builder gives the possibility to create a new query based on search terms, such as: Title, Author, Text Word, etc. The field subject can be specified, but there is also a possibility to enter a search term covered by all fields. The query can be created by inserting search terms with their corresponding field by pressing 'Enter' or to simply press the 'Add' button. After entering the first element, it unlocks the possibility to add an addition to the next element, such as AND, OR or NOT. While it is possible to create a query from scratch, GAPS also gives the availability to insert an existing query into the text field by pasting or writing it.

1.2. Specify gene symbols

This component gives the ability to include or exclude gene symbols from the search. This can be done in two ways:

  • Insert gene symbols into the text field. This can be done by inserting gene symbols into the text field. The genes need to be separated with a comma-sign.
  • Load gene symbols from an existing file. This can be done by loading a text file from your computer. The structure of the file needs to be gene symbols separated on every line. The gene symbols cannot contain characters other than letters of the alphabet and numbers, genes containing other symbols (such as commas or slashes) will not be included. 
Important While using the text field and loading a file simultaneously as input, the text field will be prioritized, and the loaded file input will not be used. Also make sure to not use any other symbols than alphabetic letters and numbers.

1.3. Optional options

In this section it is possible to specify a date range in between wanted articles are published. This can be done by specifying the publication date before and after. Pressing the calendar-icon gives the possibility to select a date from the calendar. Selecting a date can also be done by clicking on the text bar and typing in the desired dates.

1.4. Run the query

After inserting all the needed search terms, there is a possibility to clear all the input at once by pressing the 'Clear all input' button. If all search terms are filled in correctly, the search for articles can be initiated by pressing the button 'Search genes'. This will change the Query Builder page into the Results page. There is also an option to open the results in a new tab, by pressing the switch 'Open search in new tab'.
